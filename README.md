# Launger web-app
## Setup
This project requires Node, so if it's not installed yet, install the latest version from https://nodejs.org/en/download/.
### 1. Install node modules
Run the following in your terminal:
```bash
npm install
```
### 2. Copy .env variables
Run the following in your terminal:
```bash
cp .env.default .env
```
### 3. Start the app
Run the following in your terminal:
```bash
npm start
```
